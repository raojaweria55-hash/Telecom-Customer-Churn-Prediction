# Telecom Customer Churn Prediction

## Project Overview
This project is part of the SkillsHunger AI Internship Task 02. The objective is to predict customer churn using machine learning models and generate business insights.

## Dataset
Telco Customer Churn Dataset (Kaggle)

## Models Used
- Naive Bayes
- Support Vector Machine (SVM)
- Gradient Boosting

## Evaluation Metrics
- Precision
- Recall
- F1-Score

## Model Interpretability
SHAP was used to explain model predictions and identify top churn drivers.

## Key Business Insights
- Month-to-month contract customers have higher churn.
- Customers with low tenure are more likely to leave.
- Lack of online security services increases churn risk.
